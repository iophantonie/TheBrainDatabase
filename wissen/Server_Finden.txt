Dann ist dass Hier die "IOP Server finden" Vorgehensweise:
1. Überprüfe ob du auf dem Reiter "Unofficial/Inoffiziell" stehst.
2. Überprüfe, ob Passwortgeschützte Server "ausgehakt" sind.
3. Überprüfe ob bei "Spieler Server zeigen" ein Häkchen gesetzt wurde.
4. Suche in der Suchleiste nach "Infinity of Phoenix"
